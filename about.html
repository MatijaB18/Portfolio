What is

The Open Group's registered brand Unix (formally UNIX) refers to a family of computer operating systems and tools that comply with The Open Group Base Specification, Issue 7. (also known as POSIX.1-2008 or IEEE Std 1003.1 - 2008). An operating system provider must pay a licensing fee and annual trademark royalties to The Open Group in order to use the Unix brand. OS X (Apple), Solaris (Oracle), AIX (IBM), IRIX (SGI), and HP-UX are all officially licensed Unix operating systems (and their vendors) (Hewlett-Packard) 

Why is it so popular?

Unix was created using a high-level programming language (C) rather than a platform-specific assembly language, allowing it to run on a variety of computers. Unix was also created as a self-contained software system that included the operating system, development environment, tools, documentation, and editable source code.
These fundamental elements aided Unix and its derivatives in becoming an important teaching and learning tool in academic contexts, as well as widespread use and subsequent development in business settings.

Industry

Unix-based systems can be found in a variety of professional settings, running critical applications in high-availability infrastructures. Unix is an excellent platform for utility servers like DNS, DHCP, and Web servers. Its lean architecture saves resources for the apps you want to run rather than using them to boot the operating system. For programs that do not require direct user contact with the operating system, Unix provides a robust and secure platform. 

Education

Unix-based systems may be found all across the academic world, where they are used to teach students both basic and advanced computing topics. Unix is an excellent platform for teaching system administration as well as basic computing ideas. This has the extra benefit of allowing students to learn in a setting that is more similar to what they would find in the real world. Unix also comes with built-in native tools and functions, making it the system of choice for teaching programming and scripting in a variety of languages without the need for cumbersome development environments and system emulators.
